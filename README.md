# BaHaSpider
巴哈姆特動畫瘋爬蟲 with熱度排行  
抓取巴哈姆特動畫瘋資料，並根據評分或觀看人數製作排行榜，輸出成HTML表格  

#Usage  
Crawl with default number of animations:  
python BaHaSpider.py  
  
Crawl with assigned number of animations:  
python BaHaSpider.py -r 34000
